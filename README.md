# nan-api-chinese
Node编写C++扩展的Nan中文API文档，Nan是对于V8的一些API的二次抽象，旨在解决编写Node插件时遇到的跨v8版本的API兼容性问题。

并且其简化了部分的API调用形式，帮助大家更容易开发跨Node版本（0.8, 0.10, 0.12, 1, 4, 5, 6 和 7）的插件。
